Useful features

With this API you can...

Query post code information

Retrieve information regarding any post code. Fetch town, state and city; along other useful fields.

Validate address data

Use the API to validate a given town, city, state and post code combination. Find invalid fields.

Choose how you want to use it

Connect via an HTTP client or use the OpenAPI standard. Or even host the API on your own server.
